# IoT-Sensorer
Vi har arbetat med att configurera en Raspberry Pi Pico till en DHT11-sensor som läser in temperatur- och luftfuktighetsdata, vilket skickas vidare till ThingSpeak med hjälp av MicroPython. För att visualisera datan på ett ett graf har ThinkSpeak använts. Temperaturen och luftfuktigheten visas samtidigt i realtid vilket är väldigt fördelaktigt. Programmeringspråket MicroPython har används. Molntjänsten ThinkSpeak har används för att visualisera sensordata. Programmet Thonny har används för att köra programmet.
