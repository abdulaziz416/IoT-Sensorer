# IoT-Sensorer
Vi har arbetat med att configurera en Raspberry Pi Pico W till en DHT11-sensor som läser in temperatur- och luftfuktighetsdata, vilket skickas vidare till ThingSpeak med hjälp av MicroPython. För att visualisera datan på ett ett graf har ThinkSpeak använts. Temperaturen och luftfuktigheten visas samtiding i realtid
