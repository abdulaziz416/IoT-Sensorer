# Smart Väderstation - IoT Projekt

Ett komplett IoT-system för att mäta och visualisera väderdata med Raspberry Pi Pico W.

## Funktioner

- Realtidsmätning av temperatur, fuktighet och ljusnivå
- Automatisk dataöverföring till molnbackend
- Webbaserat dashboard för visualisering
- Historisk dataanalys
